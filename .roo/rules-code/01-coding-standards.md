# Python Coding Standard

This project using python 3.11+ 

1. If you logging, always use logger from loguru
2. Write proper comments
3. Error handling must be done properly.
4. Type-annotate everything; include concise docstrings on public funcs/classes.
5. Must do the imports exactly before write code.
6. When calling function. Prefer keyword arguments (e.g., func(arg1=val1, arg2=val2)) for clarity and readability, unless positional use is absolutely necessary. 

# Typescript coding standard

1. Prefer using shadcn/ui
2. ReactQuery is v5 so some callbacks is deprecated
3. Keep the styling very simple.
